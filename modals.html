Tic Tac Toe

Description: A class project to build a GUI Tic Tac Toe. The program has a feature to play against the computer option that performs behaviors based on the status of the game. Impossible will choose 100% best possible move, Hard has a 25% chance to choose the "best move." Anything else is random if the position is available. I will be porting this to Android.

  • Date: February 2015
  • Current Status: Complete
  • Experience Gain: Java, Android

Skyrim Mod: Parallaxis

Description: Skyrim is an extensive single player 3D RPG created by Bethesda Softworks that allows users to create content to share or use themselves.
