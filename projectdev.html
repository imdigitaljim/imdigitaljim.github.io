  • M-Tree: November 2016 - A competitive class project to query a huge data set to find the closest inversion distance among 32-dimensional data rows with strict memory constraints and time requirements. The constraints were a preprocessing phase was allowed that required < 1 minute of processing and a query phase which total running RAM (excluding executable and libraries) < 1MB. The query running time constraints were relative the solutions of peers' performance.
