Project Development

Python

  • Interactive Dance: April 2016 - A research project that collaborates with the Florida State University's School of Dance to develop a projected interactive environment using Computer Vision with Python and Unity3D/C#. This research was accepted into the Seventh International Conference on Computational Creativity.

This uses a Markov Chain algorithm that has been trained with a corpus consisting of Lord of the Rings and a roughly equal amount of Harry Potter.   • K-Means Clustering: December 2015 - A class project to demonstrate understanding of k-means and complete linkage clustering.

C/C++

  • WAV Steganography: March 2016 - A research project to maximize the steganographic capacity of Microsoft RIFF 16-bit PCM WAV files complemented with a shell tool for embedding files and AES encrypted steganograpic file systems.
  • Chat Application: February 2016 - A class project to develop a client/server model chat server application to provide a broadcasting group chat service.
  • FAT File System: December 2015 - A class project to build a mounting tool for linux. This will read, write, and navigate through a FAT file system image.

  • Sorting Optimization: November 2015 - A class project to demonstrate understanding of various sorting algorithms with a bonus objective of developing a highly optimized sort.
  • Linux Shell: October 2015 - A class project to design a shell system for the terminal interface of Linux that operates similarly to the Bash shell.
  • Cache Simulator: April 2015 - A class project to develop a simulation of data caching using LRU replacement with a write-back and write-allocate policy.
  • Game Engine Development: August 2015 -A work in progress of game engine components such as DirectX development.
  • MIPS Assembler: February 2015 - A class project to build an assembler for a subset of MIPS. The program will read in from standard input an instruction set in MIPS and encode it into machine code. The file is intended for unix stdio redirection.
  • Path Searching: February 2015 - A class project to design a airline flight routing application using BFS implemented with a queue.
  • MIPS Simulator: February 2015 - A class project to build a simulator for a subset of MIPS. The program will read a machine code file from argv[1] and decode it into MIPS instructions and executes. A log file is saved with the decoded instructions, the state of the registers and data memory for each increment of the program counter.
