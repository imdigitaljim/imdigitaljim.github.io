Skyrim Mod: Parallaxis

Project Description : Skyrim is an extensive single player 3D RPG created by Bethesda Softworks that allows users to create content to share or use themselves. This content can be anything from creating equipment with better stats to a fully developed environment within the game. This mod is just additional play content using existing game art and models with my addition, utilizing their game engine, being scripting and content design and implementation with a few items to earn and a few foes to vanquish.
